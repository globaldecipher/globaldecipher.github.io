# Easiest Way To Publish TGD Content

Use the GitHub form. You do not need to touch folders, code, or article settings.

1. Open the repository Issues page.
2. Click **New issue**.
3. Choose **New TGD content**.
4. Fill the boxes.
5. Click **Submit new issue**.
6. Wait one or two minutes.
7. The article appears on the website automatically.

## Which Type Should I Pick?

- **Monthly report**: for monthly terrorism and counter-terrorism assessments.
- **Daily article**: for normal news or analysis articles.
- **Daily incident log**: for daily monitoring and incident lists.
- **Opinion**: for opinion pieces.
- **Profile**: for people, groups, or organisations.

## Article Body Tips

Use simple headings like this:

```md
## Executive summary

Write here.

## What happened

Write here.

## Why it matters

Write here.
```

For tables, paste this style:

```md
| Location | Incident | Casualties |
| --- | --- | --- |
| Bannu | Short description | 0 killed, 0 injured |
```

For PDFs, paste a public PDF link in the **PDF link** box. The website article will include a download link automatically.
